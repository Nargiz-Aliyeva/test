    Html nədir

    HyperText Markup Language və ya HTML veb brauzerdə nümayiş etdirilmək üçün nəzərdə tutulmuş sənədlər üçün standart işarələmə dilidir . Bu veb məzmunun mənasını və strukturunu müəyyən edir . Buna tez-tez Cascading Style Sheets (CSS) kimi texnologiyalar və JavaScript kimi skript dilləri kömək edir . Veb-brauzerlər HTML sənədlərini veb serverdən və ya yerli yaddaşdan qəbul edir və sənədləri multimedia veb-səhifələrində təqdim edir . HTML veb-səhifənin strukturunu semantik olaraq təsvir edir və ilkin olaraq onun görünüşü üçün işarələri ehtiva edir.

    HTML elementləri HTML səhifələrinin tikinti bloklarıdır. HTML konstruksiyaları ilə şəkillər və interaktiv formalar kimi digər obyektlər göstərilən səhifəyə daxil edilə bilər. HTML, başlıqlar, paraqraflar, siyahılar, keçidlər , sitatlar və digər elementlər kimi mətnin struktur semantikasını ifadə etməklə strukturlaşdırılmış sənədlər yaratmaq üçün bir vasitə təqdim edir . HTML elementləri bucaqlı mötərizələrdən istifadə edərək yazılan teqlərlə təsvir olunur . kimi etiketlər və birbaşa səhifəyə məzmun təqdim edir. və kimi digər etiketlər

    əhatə edir və sənəd mətni haqqında məlumat verir və alt element teqlərini ehtiva edə bilər. Brauzerlər HTML teqlərini göstərmir, lakin onlardan səhifənin məzmununu şərh etmək üçün istifadə edirlər.

    HTML veb səhifələrin davranışına və məzmununa təsir edən JavaScript kimi skript dilində yazılmış proqramları yerləşdirə bilər. CSS-in daxil edilməsi məzmunun görünüşünü və tərtibatını müəyyən edir.
